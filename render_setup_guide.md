# Настройка сервиса на Render

## 1. Создание аккаунта на Render

1.  Перейдите на [https://render.com/](https://render.com/).
2.  Нажмите "Sign Up" и выберите способ регистрации (GitHub, Google, Email).
3.  Завершите процесс регистрации.

## 2. Создание Web Service

1. Войдите в [Render Dashboard](https://dashboard.render.com/).
2.  Нажмите "New +" в левом верхнем углу.
3.  Выберите "Web Service".

## 3. Подключение к репозиторию

1.  Выберите "Build and deploy from a Git repository".
2. Если вы зарегистрировались через GitHub, Render уже будет иметь доступ к вашим репозиториям. Выберите нужный репозиторий из списка.
3.  Если вы зарегистрировались по email, вам нужно будет подключить GitHub вручную через "Connect Git provider".
4.  Укажите URL вашего репозитория на GitHub, если он не был выбран автоматически.

## 4. Настройка сервиса

1.  **Service Group**: Укажите название для вашей группы сервисов (например, `tiktok-bot`).
2. **Service Name**: Укажите имя сервиса (например, `tiktok-bot-service`).
3.  **Environment**: Выберите `Python`.
4.  **Runtime**: Выберите `python-3.1:bullseye` или более новую версию, совместимую с зависимостями.
5.  **Build Command**: Укажите `pip install -r requirements.txt`.
6.  **Start Command**: Укажите `python bot.py`.
7.  **Branch**: Укажите ветку, из которой будет производиться деплой (например, `main`).
8.  **Region**: Выберите регион, ближайший к вашей аудитории (например, `Frankfurt` или `Oregon`).

## 5. Настройка переменных окружения

1.  В разделе "Environment Variables" добавьте следующие переменные:
    *   `BOT_TOKEN`: Токен вашего Telegram-бота (полученный от @BotFather).
    *   `CHANNEL_ID`: ID Telegram-канала, куда будут публиковаться видео.
    *   `ADMIN_ID`: ID вашего пользователя Telegram (для административных функций).
    *   `POSTING_INTERVAL_MINUTES`: Интервал публикации видео в минутах (опционально, по умолчанию 60).
    *   `ms_token`: (Опционально) Ваш `ms_token` для TikTok API, если требуется.

2.  Нажмите "Add Environment Variable" для каждой переменной.

## 6. Создание сервиса

1.  Проверьте все настройки.
2.  Нажмите "Create Web Service".

## 7. Мониторинг деплоя

1.  После создания сервиса вы попадете на страницу мониторинга.
2.  Render автоматически начнет процесс сборки и деплоя.
3.  Вы можете отслеживать прогресс в реальном времени.
4.  После успешного деплоя сервис будет запущен.

## 8. Проверка работы

1.  Перейдите на вкладку "Manual Deploy" или "Automatic Deploys", чтобы увидеть логи.
2.  Убедитесь, что бот запустился без ошибок.
3.  Проверьте, что бот публикует видео в канал.
